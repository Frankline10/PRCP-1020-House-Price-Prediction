#  House Price Prediction System

An end-to-end Machine Learning project that predicts house prices based on various features using advanced regression techniques. The project includes data preprocessing, feature engineering, model training, evaluation, and deployment using a web application.

---

## 📌 Project Overview

This project aims to build a robust and accurate house price prediction model. It leverages real-world housing data and applies machine learning techniques to estimate property prices based on key features such as location, size, and amenities.

The final model is deployed as an interactive web application for user-friendly predictions.

---

## 🚀 Features

* 🔹 End-to-end ML pipeline implementation
* 🔹 Data cleaning and preprocessing
* 🔹 Feature engineering and selection
* 🔹 Model training and evaluation
* 🔹 Ridge Regression as the final model
* 🔹 Model comparison with multiple algorithms like Gradient Descent Boosting algorithm and Random Forest algorithm.
* 🔹 Interactive web app using Streamlit
* 🔹 Real-time price prediction

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Libraries:** Pandas, NumPy, Scikit-learn
* **Visualization:** Matplotlib, Seaborn
* **Modeling:** Ridge Regression, Gradient Descent Boosting, Random Forest algorithm.
* **Deployment:** Streamlit

## 📈 Model Performance

* **Algorithm Used:** Ridge Regression
* **Evaluation Metrics:**

  * R² Score: 0.846051
  * RMSE: 0.155931

> Ridge Regression was selected based on its better generalization performance and lower error compared to other models.
